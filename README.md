# Simple JSON Rename Directory Shell
Shell Script for rename and move directories using JSON files.

## Prerequisites
You need there.

* jq

### jq
https://stedolan.github.io/jq/

#### Install
Homebrew
```
$ brew install jq
```

## How to use
```
$ sh rename.sh [JSON] [SRC] [DEST]
```

* JSON: JSON file ( .json )
* SRC: Source directory
* DEST: Destination directory
